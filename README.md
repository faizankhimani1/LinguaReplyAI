# LinguaReply AI 🌍

**Translate Smarter. Reply Faster.**

An AI-powered translation platform that instantly translates text between 100+ languages while providing intelligent reply suggestions to help you communicate faster and more effectively.

## 🚀 Features

### Core Translation Features
- **Real-time Translation** - Auto-translates as you type (700ms debounce)
- **100+ Languages** - Supports major world languages including Hindi, Gujarati, Chinese, Japanese, Arabic, and more
- **Auto Language Detection** - Automatically identifies source language
- **One-click Swap** - Instantly swap source and target languages

### Smart Communication Tools
- **AI Reply Suggestions** - Context-aware replies generated in your target language
- **Voice Input** - Speech-to-text translation
- **Text-to-Speech** - Listen to translations aloud with language-specific voices
- **Copy & Share** - Easy copy to clipboard functionality

### Productivity Features
- **Translation History** - Last 50 translations saved locally
- **Search History** - Find past translations quickly
- **Favorites** - Bookmark important translations
- **Character Counter** - Track input/output length

## 🛠️ Tech Stack

- **Frontend**: React 19 + TypeScript
- **Build Tool**: Vite 7
- **Styling**: Tailwind CSS v4 + Custom Glassmorphism
- **State Management**: Zustand (with localStorage persistence)
- **Animations**: Framer Motion
- **Forms**: React Hook Form
- **Notifications**: Sonner (Toast)
- **Icons**: Lucide React
